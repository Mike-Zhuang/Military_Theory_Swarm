# 实验指标与记录模板

本文件用于记录每次实验配置与指标结果，确保展示结论可复现。

## 1. 核心指标定义

1. 覆盖率（Coverage）
- 定义：已被至少一个 agent 访问的网格数 / 总网格数
- 越高越好

2. 任务完成率（Task Completion Rate）
- 定义：被标记为完成的目标数 / 总目标数
- 越高越好

3. 平均响应时间（Average Response Time）
- 定义：目标首次被分配到首次到达阈值的平均步数（或秒）
- 越低越好

4. 干扰鲁棒性保持率（Robustness Retention）
- 定义：
  - 在丢包率 $p=0$ 时的指标为 $M_0$
  - 在丢包率 $p>0$ 时的指标为 $M_p$
  - 保持率 $R=\frac{M_p}{M_0}$
- 越高越好

5. 感知质量（ML）
- 准确率、F1、混淆矩阵
- 误检率变化对任务完成率的影响曲线
